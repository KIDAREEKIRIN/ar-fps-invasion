# AR FPS — 침공 🎯

휴대폰/PC 카메라 영상을 **배경**으로 깔고, 그 위에 몰려오는 3D 드론을 격추하는 웹 기반 AR FPS 게임입니다.
별도 설치 없이 브라우저만으로 동작합니다 (Three.js + WebGL + getUserMedia + DeviceOrientation).

## 플레이 방법
- 📷 카메라 영상이 실시간 배경으로 깔립니다 (후면 카메라 우선)
- 🧭 **조준 방식 선택** — 시작 화면/일시정지에서 **자이로**(기기 기울여) ↔ **터치**(드래그) 전환, 선택은 자동 저장
- 🎯 화면을 **탭/클릭**하거나 **발사 버튼**(스페이스바)으로 사격
- ⟳ **재장전 버튼**(R 키), 탄약 소진 시 자동 재장전
- ⏸ **일시정지 버튼**(ESC), 탭을 벗어나면 자동 일시정지
- 👾 사방에서 날아오는 적을 막아내며 웨이브를 버티세요.

## 태블릿 (iPad / 갤럭시탭) 조작
태블릿은 크고 무거워 "기기를 통째로 돌려" 조준하기 불편할 수 있어, **두 가지 조준 방식**을 지원합니다.

| 동작 | 자이로 모드 | 터치 모드 (태블릿 권장) |
|---|---|---|
| 조준 | 기기를 기울여 둘러보기(AR) | 화면 **한 손가락 드래그** |
| 발사 | 화면 **탭** 또는 우측 하단 **발사 버튼** | 우측 하단 **발사 버튼** (드래그로 조준 중에도 OK) |
| 재장전 | 좌측 하단 **⟳ 버튼** | 좌측 하단 **⟳ 버튼** |
| 정면 보정 | 좌상단 **🧭 정면 버튼**(C 키) — 지금 보는 쪽을 정면으로 | — |
| 일시정지 | 우상단 **⏸ 버튼**(ESC) | 우상단 **⏸ 버튼** |

- **자이로 모드**: 첫 센서 신호 시 자동으로 정면이 보정되며, 시야가 틀어지면 **🧭 정면** 버튼으로 언제든 재보정.
- **자이로 감도 1x/1.5x/2x**: 앉아서/태블릿으로 360° 둘러보기가 힘들면 감도를 올리세요. 작은 움직임으로 넓게 조준됩니다(기본 1.5x).
- **터치 모드**: 자이로가 있어도 강제로 끄고 드래그 조준만 사용 — 거치해 두고 두 손가락(한 손은 드래그, 한 손은 발사 버튼)으로 편하게 플레이.
- **⛶ 전체화면**은 별도 버튼으로 토글합니다. 카메라/자이로 **권한 팝업과 충돌하지 않도록 시작 흐름과 분리**되어 있으니, 권한을 먼저 허용한 뒤 전체화면을 켜세요.
- 가로/세로 화면 모두 대응하며, 좌우 하단 버튼 배치로 가로 거치 시 양손 엄지 조작이 편합니다.
- 핀치 줌·스크롤·당겨서 새로고침은 비활성화되어 게임 조작과 충돌하지 않습니다.

## 무기 (3종 교체 — 하단 무기 버튼 / 1·2·3 / E)
- 🔫 **권총** — 균형형, 단발, 탄창 12
- 💥 **산탄총** — 1발에 펠릿 6개 부채꼴, 근거리 강함, 탄창 6, 느린 연사
- 🔥 **기관단총** — 빠른 **자동 연사**(누르고 있기), 탄창 30, 약한 탄퍼짐
- 강화는 **모든 무기에 공통 적용**(피해/탄창/재장전/연사 보너스), 무기별 탄약은 따로 관리됩니다.

## 적 종류 & 시스템
- 🟢 **정찰병(scout)** — 빠르고 약함(정사면체) / 🔴 **드론(drone)** — 기본 자폭형(팔면체)
- 🟠 **돌격병(brute)** — 느리지만 체력 3, 큰 피해(십이면체) / 🟣 **포탑(turret)** — 거리에서 6발 사격 후 **돌격 자폭**(큐브, 탄환 요격 가능)
- 🟩 **분열체(splitter)** — 처치 시 정찰병 2기로 분열
- 🔵 **방패병(shield)** — **약점(코어) 명중으로만** 피해, 일반 탄은 차단 → 정밀 조준 빌드와 연결
- 👑 **보스** — **5웨이브마다 등장**, 높은 체력 + **탄막(부채꼴)**, 전용 HP 바, 격파 시 대량 점수 + 확정 드롭
- 🎯 **약점(코어) 명중 = 치명타** — 운이 아니라 **조준 실력**으로 치명타 (정밀 조준 강화로 약점 범위 확대)
- ⚡ **궁극기** — 처치로 충전 게이지를 채우고, 가득 차면 발동(좌하단 버튼/Q)해 **화면의 적 일소 + 충격파**
- 🧬 **웨이브 사이 강화 선택(로그라이트)** — 매 클리어 시 3개 중 1개:
  - 고출력 탄·확장 탄창·신속 재장전·속사·강화 장갑·정밀 조준·흡수 코어·현상금
  - **관통탄**(탄 관통)·**폭발탄**(처치 시 연쇄 폭발)·**과충전**(궁극기 가속)·**✨치명 특화**(희귀, 치명타 배율↑)
- ⚡ **콤보** — 연속 처치로 점수 배율 최대 x3 (피격 시 초기화)
- 🎁 **파워업 드롭** — 체력팩(+25) 또는 오버드라이브(6초 무한 연사), 자동 회수
- 👑 **엘리트(정예)** — 일부 적이 더 크고 튼튼한 금색 링으로 등장(체력 ×3), 점수·궁극기 충전↑, **확정 드롭**
- 🎵 **동적 BGM(레이어)** — 강도(웨이브+보스+적 수)에 따라 베이스→패드/5도 화음→하이햇→아르페지오 레이어가 점층. 무기별 발사음 + **피치 변주**(명중음은 콤보가 오를수록 음정↑) (음소거로 함께 on/off)
- 📊 **명중률·최대 콤보 통계** — 게임오버 화면에 표시
- 🛠️ **무기별 전용 강화** — 권총(전용 피해)·산탄총(펠릿+3)·기관단총(탄퍼짐↓·탄창+10), 일반 강화와 별도로 누적
- 🏆 **랭킹 Top 10** — 게임오버 시 이름 입력 → 점수·웨이브 저장·표시(내 기록 하이라이트), **시작 화면에 Top 5** 노출. 기본 로컬(localStorage), **온라인 글로벌 랭킹(Supabase)도 클라이언트 연동 완료** — `game.js` 상단의 `SUPABASE_URL`/`SUPABASE_KEY`만 채우면 자동으로 글로벌 모드로 전환(미설정 시 로컬 폴백).
- 🎮 **배경 모드 2종** — **AR**(카메라 배경) / **아레나**(카메라 없이 3D 환경: 솔리드 배경 + 네온 그리드 바닥). 시작 화면에서 선택, 저장됨.
- 🌌 **구역(AR 테마)** — 3웨이브마다 구역 전환: 사이버 그리드/용암 지대/심해/우주 정거장/사막 폭풍. 화면 색감 틴트 + 부유 파티클 + 거리 안개가 카메라 배경 위에서 분위기를 바꿉니다(카메라 배경은 유지).
- 🧭 **화면 밖 위협 표시기** · 🔊 **음소거** · 🏆 **최고 점수** 자동 저장(localStorage)
- 🛡️ 동시 적 수 상한·속도 상한으로 후반 난이도 폭주 방지

## 온라인 글로벌 랭킹 설정 (선택 — 기본은 로컬)
클라이언트 연동은 완료돼 있고, Supabase 프로젝트만 연결하면 자동으로 글로벌 모드가 됩니다.

1. Supabase 프로젝트의 SQL 에디터에서 아래 실행:
   ```sql
   create table public.scores (
     id bigint generated always as identity primary key,
     name text not null check (char_length(name) <= 16),
     score int not null check (score >= 0 and score < 100000000),
     wave int not null default 1 check (wave between 1 and 10000),
     created_at timestamptz not null default now()
   );
   alter table public.scores enable row level security;
   create policy "public read"   on public.scores for select using (true);
   create policy "public insert" on public.scores for insert with check (true);
   create index scores_score_idx on public.scores (score desc);
   ```
2. `game.js` 상단의 `SUPABASE_URL`, `SUPABASE_KEY`(anon/publishable 공개키)를 채우면 끝.
   비워 두면 자동으로 로컬 랭킹으로 폴백합니다. (게임오버 랭킹 제목이 `(로컬)`/`(글로벌)`로 표시)

> 익명 insert라 누구나 등록 가능(치팅 여지). 캐주얼 랭킹엔 일반적이며, 필요하면 레이트리밋/서버 검증을 추가하세요.

### 휴대폰에서 테스트하려면
모바일의 카메라/자이로 권한은 **HTTPS**에서만 허용됩니다. 두 가지 방법:
1. **ngrok / cloudflared 터널**
   ```bash
   npx cloudflared tunnel --url http://localhost:8765
   # 또는: ngrok http 8765
   ```
   발급된 `https://...` 주소를 휴대폰에서 접속.
2. 정적 호스팅(Netlify, GitHub Pages, Vercel)에 업로드 후 접속.

> iOS 사파리는 첫 "게임 시작" 버튼 탭 시 자이로/카메라 권한을 요청합니다(사용자 제스처 필요).
