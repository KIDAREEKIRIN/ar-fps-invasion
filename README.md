# AR FPS — 침공 🎯

휴대폰/PC 카메라 영상을 **배경**으로 깔고, 그 위에 몰려오는 3D 드론을 격추하는 웹 기반 AR FPS 게임입니다.
별도 설치 없이 브라우저만으로 동작합니다 (Three.js + WebGL + getUserMedia + DeviceOrientation).

## 플레이 방법
- 📷 카메라 영상이 실시간 배경으로 깔립니다 (후면 카메라 우선)
- 🧭 **조준 방식 선택** — 시작 화면/일시정지에서 **자이로**(기기 기울여) ↔ **터치**(드래그) 전환, 선택은 자동 저장
- 🎯 화면을 **탭/클릭**하거나 **발사 버튼**(스페이스바)으로 사격
- ⟳ **재장전 버튼**(R 키), 탄약 소진 시 자동 재장전
- ⏸ **일시정지 버튼**(ESC), 탭을 벗어나면 자동 일시정지
- 👾 사방에서 날아오는 적을 막아내며 웨이브를 버티세요.

## 태블릿 (iPad / 갤럭시탭) 조작
태블릿은 크고 무거워 "기기를 통째로 돌려" 조준하기 불편할 수 있어, **두 가지 조준 방식**을 지원합니다.

| 동작 | 자이로 모드 | 터치 모드 (태블릿 권장) |
|---|---|---|
| 조준 | 기기를 기울여 둘러보기(AR) | 화면 **한 손가락 드래그** |
| 발사 | 화면 **탭** 또는 우측 하단 **발사 버튼** | 우측 하단 **발사 버튼** (드래그로 조준 중에도 OK) |
| 재장전 | 좌측 하단 **⟳ 버튼** | 좌측 하단 **⟳ 버튼** |
| 정면 보정 | 좌상단 **🧭 정면 버튼**(C 키) — 지금 보는 쪽을 정면으로 | — |
| 일시정지 | 우상단 **⏸ 버튼**(ESC) | 우상단 **⏸ 버튼** |

- **자이로 모드**: 첫 센서 신호 시 자동으로 정면이 보정되며, 시야가 틀어지면 **🧭 정면** 버튼으로 언제든 재보정.
- **터치 모드**: 자이로가 있어도 강제로 끄고 드래그 조준만 사용 — 거치해 두고 두 손가락(한 손은 드래그, 한 손은 발사 버튼)으로 편하게 플레이.
- 가로/세로 화면 모두 대응하며, 좌우 하단 버튼 배치로 가로 거치 시 양손 엄지 조작이 편합니다.
- 핀치 줌·스크롤·당겨서 새로고침은 비활성화되어 게임 조작과 충돌하지 않습니다.

## 적 종류 & 시스템
- 🟢 **정찰병(scout)** — 빠르고 약함 / 🔴 **드론(drone)** — 기본 자폭형
- 🟠 **돌격병(brute)** — 느리지만 체력 3, 큰 피해 / 🟣 **포탑(turret)** — 거리를 두고 **유도탄 발사**(탄환도 쏴서 요격 가능)
- ⚡ **콤보** — 연속 처치로 점수 배율 최대 x3 (피격 시 초기화)
- 💥 **치명타** — 18% 확률로 2배 피해·점수
- 🎁 **파워업 드롭** — 격추 시 확률적으로 체력팩(+25) 또는 오버드라이브(6초 무한 연사) 등장, 자동 회수
- 🏆 **최고 점수** 자동 저장(localStorage)

## 실행 (중요: 카메라·자이로는 보안 컨텍스트 필요)
`file://`로 직접 열면 ES 모듈/카메라가 동작하지 않습니다. **로컬 서버**로 띄우세요.

```bash
cd /Users/keonwookim/Desktop/side_project/010
python3 -m http.server 8765
```
그 후 브라우저에서 `http://localhost:8765` 접속.

### 휴대폰에서 테스트하려면
모바일의 카메라/자이로 권한은 **HTTPS**에서만 허용됩니다. 두 가지 방법:
1. **ngrok / cloudflared 터널**
   ```bash
   npx cloudflared tunnel --url http://localhost:8765
   # 또는: ngrok http 8765
   ```
   발급된 `https://...` 주소를 휴대폰에서 접속.
2. 정적 호스팅(Netlify, GitHub Pages, Vercel)에 업로드 후 접속.

> iOS 사파리는 첫 "게임 시작" 버튼 탭 시 자이로/카메라 권한을 요청합니다(사용자 제스처 필요).
